# Rock-Paper-Scissor-Game
A fun and interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript.
This project lets you play against the computer with instant results and a simple, responsive UI.

**🚀 Features**

🎮 Player vs Computer gameplay

🤖 Computer generates random choices

🖥️ Responsive design for desktop & mobile

🏆 Shows winner for each round instantly

🔄 Option to restart the game

**🛠️ Technologies Used**

HTML5 – for game structure

CSS3 – for design and responsiveness

JavaScript (ES6) – for game logic and interactivity
